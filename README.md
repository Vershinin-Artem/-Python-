**Задание №4**

**Задание №5**

То есть сначала нужно посчитать, сколько в число n поместилось целых часов, целых минут и сколько после этого осталось секунд. 

Решение:
```python

minutes = (n % 3600) //60
seconds = (n % 60)
```

Решение:
```python

sum_right = sum(a[middle:])
sum_left = sum(a[:middle])
result = sum_right == sum_left
```
